# 🔗 Crypto Wallet REST API

REST API sederhana untuk membuat wallet Ethereum baru dan cek saldo menggunakan Flask dan Web3.py.

---

## 🛠 Cara Menjalankan

1. Install dependensi:
   ```bash
   pip install -r requirements.txt

---
2. Ganti YOUR_INFURA_PROJECT_ID di app.py dengan project ID Infura kamu.

---

3. Jalankan aplikasi :
    ```bash
   python app.py

---

4. Endpoint yang tersedia:

    POST /create_wallet
    Membuat wallet baru. Response: alamat dan private key.

    GET /get_balance?address=0x...
    Mengecek saldo ETH dari alamat wallet
